# Experiments

Use this directory to keep the project reproducible and easy to audit.

## Files

- `experiment_log.csv`: one row per meaningful training run
- `submission_log.csv`: one row per Kaggle submission

## Suggested Practice

- create an experiment entry before or immediately after each run
- link every submission back to an experiment ID
- keep notes short, concrete, and focused on what changed
- record data revision IDs whenever labels are updated in 3LC
